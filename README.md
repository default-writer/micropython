# micropython

Just one more playground

## Initalization

To inialize, run commands:

### Linux

```bash
python -m pip install virtualenv
virtualenv venv
python -m pip install --upgrade pip
python -m pip install -r requirements.txt
```

### Windows (PowerShell)

```bash
python -m pip install virtualenv
virtualenv venv
python -m pip install --upgrade pip
python -m pip install -r ./requirements.txt
```
